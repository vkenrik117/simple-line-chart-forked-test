# Модифицированный пример графика recharts.org

В отличие от оригинала, подкрашивает все участки графиков значение y которых не входят в определенный интервал

## Компоненты

* ### chart-data-service.ts:
        предоставляет все функциональные данные
* ### App.tsx:
        отображает данные на графике
* ### ChartModels.ts:
        содержит модели данных
